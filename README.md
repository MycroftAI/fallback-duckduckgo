## About 
Query DuckDuckGo as a fallback when no other Skill can answer the question.

Uses the [DuckDuckGo API](https://duckduckgo.com/api) to provide information. 


## Examples 
* "Who is george washington?"
* "What is plasma?"
* "Who's madonna?"
